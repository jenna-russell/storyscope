# StoryScope Dataset

## Overview

This dataset accompanies the StoryScope paper. It contains 10,272 writing prompts, each paired with AI-generated stories from 5 LLMs, plus metadata about the original human-authored source stories from Books3. Human story text is **not included** due to copyright.

## Files

### Stories (`stories_{split}.parquet`)

Four parquet files split by dataset partition: `stories_dev.parquet` (included in repo), plus `stories_train.parquet`, `stories_val.parquet`, `stories_test.parquet` (hosted externally due to size).

| Column | Type | Description |
|--------|------|-------------|
| `prompt_id` | int | Unique prompt identifier |
| `split` | str | Dataset split: train, val, test, dev |
| `title` | str | Story title (from original human-authored work) |
| `prompt` | str | Writing prompt (reverse-engineered from human story) |
| `human_author` | str | Author of the original human-written story |
| `human_anthology` | str | Anthology/collection containing the original story |
| `human_word_count` | float | Word count of the original human story |
| `story_gpt` | str | AI story generated by GPT-5.4 |
| `story_deepseek` | str | AI story generated by DeepSeek V3.2 |
| `story_kimi` | str | AI story generated by Kimi K2.5 |
| `story_gemini` | str | AI story generated by Gemini 3 Flash |
| `story_claude` | str | AI story generated by Claude Sonnet 4.6 |

**Split sizes:**

| Split | Prompts | Purpose |
|-------|---------|---------|
| train | 7,383 | Classifier training |
| val | 1,405 | Hyperparameter tuning |
| test | 1,384 | Held-out evaluation |
| dev | 100 | Feature taxonomy discovery |

### Features (`storyscope_features.parquet`)

304 narrative features extracted for each of the ~61,600 stories (all sources x all prompts).

| Column | Type | Description |
|--------|------|-------------|
| `prompt_id` | int | Prompt identifier (joins with stories) |
| `story_title` | str | Story title |
| `source` | str | Source: human, gpt, claude, deepseek, kimi, gemini |
| `SIT_GEN_001` ... `STY_FIG_039` | str | 304 feature value columns |

Feature values are strings: categorical labels, ordinal labels, scale integers (as strings), "yes"/"no" for binary, and pipe-separated values for multi-select (e.g., `"explicit named\|retelling"`).

The 304-feature taxonomy organized by NarraBench dimension and aspect.

### Model Weights (`models/`)

Trained XGBoost classifiers:

| File | Task | Features |
|------|------|----------|
| `binary_narrative.json` | Human vs AI | Narrative only (no style) |
| `binary_full.json` | Human vs AI | All 304 features |
| `multiclass_narrative.json` | 6-way attribution | Narrative only |
| `multiclass_full.json` | 6-way attribution | All 304 features |

Load with:
```python
from xgboost import XGBClassifier
clf = XGBClassifier()
clf.load_model("data/models/binary_narrative.json")
```

## Copyright Notice

Human story text from Books3 is excluded due to copyright restrictions. The dataset includes book metadata (author name, anthology title, word count) for research reproducibility. AI-generated stories are released for research purposes.
